What is the Document Object Model(DOM)?

How do you select an element from the DOM?The Document Object Model (DOM) is a programming interface that represents HTML documents in a tree-like structure. It helps to change the content, structure, and style of the document.

How do you select an element from the DOM?

JavaScript is used to select an element from the DOM. For example: 1.document.getElementById('id') - to select element of specified id. 2.document.querySelector('.class') - Uses a class or selector. 3.document.querySelectorAll('tag')- to select all elements of the same tag.

What is the event delegation in the context of the DOM, and why it is usefull?

Event Delegation** is a technique where an event listener is added on a parent component and it handles events on its child components. This is useful because it increases performance, requires fewer event listeners, and can apply events to dynamically created components.

How do you manipulate an element's attributes and style using DOM?

Changing the properties and styles of elements using the DOM is very easy. For example, setAttribute() is used to change the attributes of an element, and the style property is used to change the style: const element = document.querySelector('div'); element.setAttribute('id', 'newId'); element.style.backgroundColor = 'blue'; element.style.fontSize = '20px'; It changes attributes and styles.
